## Car Price Prediction (Linear Models)

В данном проекте решается задача регрессии: предсказание стоимости автомобилей на основе числовых и категориальных признаков.

Основной целью является анализ эффективности линейных моделей и оценка влияния признаков на качество предсказания.

### Данные

Используется датасет автомобилей, содержащий следующие признаки:
- Make — марка автомобиля
- Model — модель (исключён из-за высокой кардинальности)
- Year — год выпуска
- Distance — пробег
- Engine_capacity — объём двигателя
- Fuel_type — тип топлива
- Transmission — тип коробки передач
- Style — тип кузова
- Price — целевая переменная

### Предобработка данных

В рамках подготовки данных были выполнены следующие шаги:

1. удаление аномальных значений:
- цена < 500 евро
- объём двигателя < 800 см³
- год выпуска < 1990

2. создание нового признака - 'car_age' = 2021 - Year

3. удаление признака Year (во избежание мультиколлинеарности)

4. удаление признака Model из-за высокой кардинальности

5. Обработка признаков:
- числовые признаки → стандартизация (StandardScaler)
- категориальные признаки:
- объединение редких категорий (RareGrouper)
- one-hot encoding (OneHotEncoder)

### Модели

В проекте исследуются следующие линейные модели:
- LinearRegression
- Ridge (L2-регуляризация)
- Lasso (L1-регуляризация)
- SGDRegressor

Модели обучаются в двух вариантах:
- только числовые признаки (num)
- числовые + категориальные (num+cat)

## Оценка моделей
Для сравнения моделей использована 5-fold кросс-валидация.

Основная метрика - RMSE (в логарифмированном пространстве)

Результаты:
- добавление категориальных признаков снизило RMSE с ~0.45 → до ~0.40
- различия между линейными моделями оказались минимальными
- стандартное отклонение метрики (~0.006–0.007) указывает на стабильность моделей

### Финальная модель

Выбрана модель LinearRegression + num+cat

Причины:
- лучшая метрика на кросс-валидации
- высокая интерпретируемость
- отсутствие значимого выигрыша от регуляризации

### Результаты на тестовой выборке

- RMSE: ~3720 евро
- MAE: ~2110 евро
- R²: ~0.76

### Анализ качества модели

Проведён визуальный анализ:
- True vs Predicted — модель улавливает общий тренд
- Residual plot — ошибка увеличивается с ростом цены
- Error distribution — большинство ошибок сосредоточено около нуля
- MAE by segment — модель хуже работает на дорогих автомобилях

### Выводы

1. основной вклад в качество модели даёт использование категориальных признаков

2. выбор конкретной линейной модели оказывает минимальное влияние

3. модель хорошо работает для массового сегмента

4. для дорогих автомобилей ошибка значительно выше
